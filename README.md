## How to run this project?

Firstly clone this project. php version must be 8.2
then open Terminal.
```bash
  composer update
```

Create a Database -- name = taskmanager, then run this command
```bash
  php artisan migrate
```

Run Laravel Project
```bash
  php artisan serve
```
